# Project Memory
- Sempre que editar ou escrever um arquivo, utiliza encoding Windows 1252;
- Linguagem AdvPL/TLPP;

## Context
- Rotina desenvolvida em AdvPL (Linguagem de programaçăo da Totvs para ERP Protheus) criada com intuito de facilitar e agilizar a análise e geraçăo do pedido de compra, solicitaçăo de compra e/ou ordem de produçăo por meio de cálculos de MRP;
- Seu objetivo é permitir que o usuário de supply chain pesquisa por produto, grupo de produto, fornecedor ou outro filtro que lhe convenha e o sistema realize os cálculos considerando as fórmulas cadastradas para apresentar sugestăo para compra, solicitaçăo de compra ou para produçăo.
